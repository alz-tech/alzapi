🎵 Music

Music Endpoints

Search songs and artists via Deezer, get album art, track info, 30-second stream previews, and full artist discographies.

GET/api/music/searchTry it →

Search for songs and artists. Returns track list with titles, artists, album art, and preview links.

Query Parameters
ParameterTypeRequiredDescription
qstringrequiredSong name, artist, or album
limitintegeroptionalNumber of results (default: 10, max: 50)
Example Request
GET
https://api.alz.name.ng/api/music/search?q=Burna+Boy&limit=10
GET/api/music/artistTry it →

Search and get an artist's profile including bio, genre, top tracks, and discography.

Query Parameters
ParameterTypeRequiredDescription
qstringrequiredArtist name
GET/api/music/track/:id

Get full details for a specific track by its Deezer track ID.

Path Parameters
ParameterTypeRequiredDescription
:idintegerrequiredDeezer track ID (from search results)
GET/api/music/artist/:id/tracks

Get top tracks from an artist by their Deezer artist ID.

Path Parameters
ParameterTypeRequiredDescription
:idintegerrequiredDeezer artist ID
GET/api/music/previewTry it →

Stream a 30-second audio preview of a track directly in the browser.

Query Parameters
ParameterTypeRequiredDescription
idintegerrequiredDeezer track ID
Returns an audio/mpeg stream. Use as <audio src="..."> or pipe it directly.
