💱 Currency & Crypto

Currency & Crypto Endpoints

Convert fiat currencies, get live exchange rates, 365-day rate history, list all supported currencies, and look up real-time cryptocurrency prices.

GET/api/currency/convertTry it →

Convert an amount from one currency to another using live exchange rates.

Query Parameters
ParameterTypeRequiredDescription
fromstringrequiredSource currency code (e.g. USD)
tostringrequiredTarget currency code (e.g. NGN)
amountnumberoptionalAmount to convert (default: 1)
Example Request
GET
https://api.alz.name.ng/api/currency/convert?from=USD&to=NGN&amount=100
Example Response
JSON
{
  "success": true,
  "from": "USD",
  "to": "NGN",
  "amount": 100,
  "result": 162500,
  "rate": 1625
}
GET/api/currency/ratesTry it →

Get all current exchange rates relative to a base currency.

Query Parameters
ParameterTypeRequiredDescription
basestringoptionalBase currency (default: USD)
GET/api/currency/list

List all supported fiat currencies with their full names and codes.

GET/api/currency/historyTry it →

Get historical exchange rate data for a currency pair over a number of days (up to 365).

Query Parameters
ParameterTypeRequiredDescription
fromstringrequiredSource currency code
tostringrequiredTarget currency code
daysintegeroptionalNumber of days of history (default: 30, max: 365)
GET/api/currency/cryptoTry it →

Get real-time cryptocurrency prices via CoinGecko. Supports multiple coins and multiple fiat currencies.

Query Parameters
ParameterTypeRequiredDescription
coinsstringrequiredComma-separated coin IDs (e.g. bitcoin,ethereum,solana)
vsstringoptionalTarget currency (default: usd)
Example Request
GET
https://api.alz.name.ng/api/currency/crypto?coins=bitcoin,ethereum,solana&vs=usd
GET/api/currency/crypto/convertTry it →

Convert a crypto amount to fiat or another crypto.

Query Parameters
ParameterTypeRequiredDescription
fromstringrequiredCoinGecko coin ID (e.g. bitcoin)
tostringrequiredTarget currency (e.g. ngn)
amountnumberoptionalAmount to convert (default: 1)
