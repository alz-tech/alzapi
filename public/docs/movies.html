🎬 Movies & TV

Movies & TV Endpoints

Search movies and TV shows, get full details with cast and trailers, and browse trending or popular lists — powered by TMDB.

GET/api/movies/searchTry it →

Search for movies and TV shows. Returns title, poster, rating, overview, release date, and TMDB ID.

Query Parameters
ParameterTypeRequiredDescription
qstringrequiredMovie or TV show name to search
Example Request
GET
https://api.alz.name.ng/api/movies/search?q=The+Dark+Knight
Example Response
JSON
{
  "success": true,
  "results": [{
    "id": 155,
    "title": "The Dark Knight",
    "overview": "Batman raises the stakes...",
    "poster": "https://image.tmdb.org/t/p/w500/...",
    "rating": 9.0,
    "release_date": "2008-07-18"
  }]
}
GET/api/movies/:idTry it →

Get full details for a specific movie or TV show by its TMDB ID, including cast, crew, trailers, and similar titles.

Path Parameters
ParameterTypeRequiredDescription
:idintegerrequiredTMDB movie or show ID (get from search results)
Example Request
GET
https://api.alz.name.ng/api/movies/155
