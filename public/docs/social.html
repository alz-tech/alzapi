📲 Social Media

Social Media Endpoints

Download videos and audio from TikTok (no watermark), Instagram, Twitter/X, and Facebook. Supports info lookup before downloading.

GET/api/social/platforms

Get a list of all supported platforms and their status.

Example Response
JSON
{
  "success": true,
  "platforms": ["tiktok", "instagram", "twitter", "facebook"]
}
GET/api/social/infoTry it →

Fetch metadata for a social media post including title, thumbnail, author, and available media formats — without downloading.

Query Parameters
ParameterTypeRequiredDescription
urlstringrequiredFull URL of the TikTok, Instagram, Twitter/X, or Facebook post
Example Request
GET
https://api.alz.name.ng/api/social/info?url=https://www.tiktok.com/@user/video/123
GET/api/social/downloadTry it →

Download a social media video or audio. TikTok downloads are watermark-free.

Query Parameters
ParameterTypeRequiredDescription
urlstringrequiredFull URL of the post to download
typestringoptionalvideo (default) or audio
Example Request
GET
https://api.alz.name.ng/api/social/download?url=https://www.tiktok.com/@user/video/123&type=video
This endpoint streams the media file. Open in browser or use as a download link. The response content type will be video/mp4 or audio/mpeg.
