🔍 Search & Wikipedia

Search & Wikipedia Endpoints

Full Wikipedia article lookup with thumbnails and extracts, DuckDuckGo instant answers, autocomplete suggestions, and random article discovery.

GET/api/search/wikiTry it →

Get a full Wikipedia article including extract, thumbnail, related links, and summary sections.

Query Parameters
ParameterTypeRequiredDescription
qstringrequiredTopic or article title to look up
Example Request
GET
https://api.alz.name.ng/api/search/wiki?q=Elon+Musk
Example Response
JSON
{
  "success": true,
  "title": "Elon Musk",
  "extract": "Elon Reeve Musk is a businessman...",
  "thumbnail": "https://upload.wikimedia.org/...",
  "url": "https://en.wikipedia.org/wiki/Elon_Musk"
}
GET/api/search/wiki/searchTry it →

Search Wikipedia and return a list of matching article titles.

Query Parameters
ParameterTypeRequiredDescription
qstringrequiredSearch query
GET/api/search/wiki/randomTry it →

Get a random Wikipedia article. Great for discovery bots and trivia features.

https://api.alz.name.ng/api/search/wiki/random
