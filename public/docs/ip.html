🌐 IP Geolocation

IP Geolocation Endpoints

Look up any IP address or domain name. Returns country, city, region, ISP, timezone, coordinates, ASN, and proxy/VPN detection.

GET/api/ipTry it →

Look up the caller's own IP address (the IP making the request). Useful for detecting the user's location automatically.

Example Request
GET
https://api.alz.name.ng/api/ip
Example Response
JSON
{
  "success": true,
  "ip": "197.210.12.45",
  "country": "Nigeria",
  "country_code": "NG",
  "city": "Lagos",
  "region": "Lagos State",
  "isp": "MTN Nigeria",
  "timezone": "Africa/Lagos",
  "lat": 6.4541,
  "lon": 3.3947,
  "proxy": false,
  "vpn": false
}
GET/api/ip/:addressTry it →

Look up any specific IP address or domain name. Resolves domain to IP automatically.

Path Parameters
ParameterTypeRequiredDescription
:addressstringrequiredIPv4 or IPv6 address, or a domain name (e.g. google.com)
Example Requests
GET
https://api.alz.name.ng/api/ip/8.8.8.8
https://api.alz.name.ng/api/ip/google.com
Both IPv4 (e.g. 8.8.8.8) and domain names (e.g. cloudflare.com) are accepted. Domain names are resolved before lookup.
