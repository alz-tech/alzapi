YouTube Endpoints

Search YouTube, fetch video metadata, stream or download audio/video as MP3 or MP4, and get direct temporary media links.

GET/api/yt/searchTry it →

Search YouTube and return a list of videos with title, thumbnail, duration, view count, and channel info.

Query Parameters
ParameterTypeRequiredDescription
qstringrequiredSearch query
limitintegeroptionalNumber of results (default: 10, max: 20)
Example Request
GET
https://api.alz.name.ng/api/yt/search?q=lofi+hip+hop&limit=5
Example Response
JSON
{
  "success": true,
  "results": [{
    "title": "lofi hip hop radio",
    "videoId": "jfKfPfyJRdk",
    "url": "https://youtu.be/jfKfPfyJRdk",
    "thumbnail": "https://i.ytimg.com/vi/...",
    "duration": "LIVE",
    "views": 1200000,
    "channel": "Lofi Girl"
  }]
}
GET/api/yt/infoTry it →

Get full metadata for a YouTube video: title, description, thumbnail, duration, formats available, and channel details.

Query Parameters
ParameterTypeRequiredDescription
urlstringrequiredFull YouTube video URL (youtu.be or youtube.com)
Example Request
GET
https://api.alz.name.ng/api/yt/info?url=https://youtu.be/dQw4w9WgXcQ
GET/api/yt/linkTry it →

Get a direct temporary download link for a YouTube video as MP3 or MP4 without triggering a download.

Query Parameters
ParameterTypeRequiredDescription
urlstringrequiredFull YouTube video URL
typestringoptionalmp3 (default) or mp4
GET/api/yt/downloadTry it →

Download a YouTube video directly. The server pipes the file to the browser, triggering a download with the correct filename and MIME type.

Query Parameters
ParameterTypeRequiredDescription
urlstringrequiredFull YouTube video URL
typestringoptionalmp3 (default) or mp4
This endpoint streams a binary file. Open it in the browser or use as a download href — do not call it as a JSON endpoint.
GET/api/yt/streamTry it →

Stream a YouTube video directly in the browser without downloading. Ideal for audio players or inline video.

Query Parameters
ParameterTypeRequiredDescription
urlstringrequiredFull YouTube video URL
typestringoptionalmp3 (default) or mp4
