AI Endpoints

Text generation using multiple AI models (GPT-4o, Mistral, Claude, Llama, Gemini) and image synthesis powered by Flux and SDXL. All free, no key needed.

GET /api/ai/chat Try it →

Ask any question and get a response from the selected AI model. Powered by Pollinations AI.

Query Parameters
ParameterTypeRequiredDescription
qstringrequiredThe question or prompt to send to the AI
modelstringoptionalModel to use: openai (default), mistral, claude, llama, gemini
systemstringoptionalSystem prompt to set the AI's behaviour. Defaults to a helpful assistant.
Example Request
GET
https://api.alz.name.ng/api/ai/chat?q=What+is+quantum+computing&model=openai
Example Response
JSON
{
  "success": true,
  "model": "openai",
  "query": "What is quantum computing",
  "response": "Quantum computing uses quantum bits (qubits)..."
}
GET /api/ai/image Try it →

Generate images from a text prompt. Returns the image directly (binary stream) or a URL depending on the request. Uses Flux and SDXL models via Pollinations AI.

Query Parameters
ParameterTypeRequiredDescription
promptstringrequiredText description of the image to generate
modelstringoptionalflux (default), flux-realism, turbo
widthintegeroptionalImage width in pixels (default: 1024)
heightintegeroptionalImage height in pixels (default: 1024)
seedintegeroptionalSeed for reproducible results
Example Request
GET
https://api.alz.name.ng/api/ai/image?prompt=a+futuristic+purple+city+at+night&model=flux
This endpoint returns the image directly as a binary stream. You can use the URL as an <img src="..."> tag or download it. No JSON wrapper.
GET /api/ai/models

List all available AI text models.

Example Response
JSON
{
  "success": true,
  "models": ["openai", "mistral", "claude", "llama", "gemini"]
}
